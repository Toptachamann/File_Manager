1. В «Приложении 1»:
- Считать из исходного файла payments_input.txt (текстовый файл с разделителями «|») данные о пяти платежах 
- Создать класс Платежи (поля customerNumber, paymentDate, amount) и 5 объектов этого класса для каждого из 5 платежей их исходного файла
- Сериализировать данные объекты в файл(ы). 
2. В «Приложении 2»:
- Создать(переработать) класс Платежи, добавив к нему поле checkNumber.
- Десериализировать объекты из файлов, полученных в «Приложении 1». 
- Внести в дополнительное поле checkNumber – случайный набор символов
- Записать полученные даные в выходной файл payments_output.txt (текстовый файл с разделителями «|»).
